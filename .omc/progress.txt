### Key Decisions

- Did NOT use `#Requires -RunAsAdministrator` — used inline elevation check instead so script can self-relaunch
- Did NOT dot-source Common.ps1 before setting execution policy (chicken-and-egg: policy may be Restricted)
  - Set-ExecutionPolicy runs as a cmdlet even in Restricted mode; subsequent dot-sourcing then works
  - In practice Setup-Dotfiles.ps1 doesn't dot-source Common.ps1 at all (avoids the dependency)
- Windows Terminal package dir glob (`Microsoft.WindowsTerminal_*`) avoids hardcoding the version hash
- PSScriptAnalyzer not installed on this machine; CI will catch any lint issues
